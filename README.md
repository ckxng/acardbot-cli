# acardbot-cli

A simple adaptive cards CLI for Teams incoming webhooks

## Usage

Install and use the script as follows:

    npm install -g @ckxng/acardbot-cli
    
    acardbot --help
    acardbot -t "title" -m "message" -w "https://webhookurl"
